TAO

TAO is a beautiful, unique, and extremely minimalist Tree Annotation Syntax which can be used in all kinds of contexts.

It is a simple alternative to XML, JSON, S-expressions and others.

I hope you have as much fun using it as I had in discovering it.

See https://tree-annotation.org/ (mirror here, not necessarily up-to-date), https://github.com/tree-annotation or contact me at dariusz.jedrzejczak.work for more details.
