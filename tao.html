TAO

TAO is a simple project with a grand vision.

It is meant to be a universal syntax for structural communication.

I imagine future evolutions of formats like JSON, HTML, XML, S-expressions, all built on TAO.

I created it because I would love if something like it would be widely available and supported.

I hope you have as much fun using it as I have in developing it.

See https://tree-annotation.org/ for more details.
